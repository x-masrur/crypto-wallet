# ◈ ZEN_VAULT

A high-fidelity, minimalist Ethereum wallet interface designed for the modern web. **Zen_Vault** prioritizes a distraction-free environment while providing robust multi-chain functionality.

---

## ⚡ Features

* **Multi-Chain Support**: Integrated RPCs for Ethereum, Polygon, Base, Arbitrum, Optimism, BNB, and Monad Testnet.
* **Secure Local Storage**: Uses `Crypto-JS` for AES-256 encryption. Your private keys never leave your browser's local storage.
* **Glassmorphic UI**: A custom-built aesthetic using Tailwind CSS and backdrop filters for a premium "frosted glass" look.
* **On-Device Generation**: Create new BIP-39 compatible mnemonic phrases or import existing private keys.
* **Real-time Tracking**: Live balance fetching and USD conversion (estimated) via Alchemy RPC nodes.
* **Responsive Drawer System**: Intuitive "Receive" and "Send" workflows designed for both mobile and desktop.

---

## 🛠️ Tech Stack

* **Logic**: [Ethers.js v6](https://docs.ethers.org/)
* **Security**: [Crypto-JS](https://cryptojs.gitbook.io/)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Typography**: JetBrains Mono
* **Icons**: Font Awesome 6
